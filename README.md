# Groupomania

## Installation

Cloner le dépot et lancez la commande:

```
npm install
```

## Fichier de configuration

Fichier dev:
Renommer .env.development.example en .env.development

Fichier production:
Renommer .env.production.example en .env.production

## Mode développement

```
npm run dev
```

### Compiler l'API

```
npm run build
```

### Mode production

```
npm run start
```

### Tuer le process daemon pm2

```
pm2 kill
```
